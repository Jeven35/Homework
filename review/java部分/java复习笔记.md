1. equals和"=="的区别

   答：默认情况下也就是从超类Object继承而来的equals方法和"=="完全等价的，都是比较内存地址，但我们可以重写equals方法，使其按照我们的需求进行比较，比如String类的equals方法，比较的是字符串的序列而不是字符串的地址。

   equals重写要满足五个规则：

   1. 自反性
   2. 对称性
   3. 传递性
   4. 一致性
   5. 对于任何非空引用值x,x.equals(null)返回false；
